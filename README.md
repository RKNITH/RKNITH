I'm a full-stack developer from Patna, India, specialising in the MERN stack with a growing focus on machine learning and cybersecurity. I build end-to-end web applications — from responsive UIs with React and Next.js to RESTful APIs with Node.js and Express — and I'm currently expanding into deep learning (CNNs, GANs) and ethical hacking fundamentals.

I care about clean code, real-world impact, and continuously levelling up across the stack.
